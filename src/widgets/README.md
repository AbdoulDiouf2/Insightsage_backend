# Widgets Module - InsightSage

Ce module gérera les composants individuels affichables sur les tableaux de bord (Cockpits).

## Fonctionnalités préméditées
- **Le `Widget Store`** : Bibliothèque centrale où l'utilisateur pourra chercher, filtrer et ajouter des widgets à son Cockpit.
- **Packs de KPIs** : Intégration de packs métiers (ex: KPI Pack DAF) pré-définissant un ensemble de widgets.
- Configuration granulaire : Stocker la position, la taille et les filtres temporels dédiés de chaque occurrence de widget attachée à un tableau de bord.
