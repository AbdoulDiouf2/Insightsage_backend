# NLQ Module - InsightSage

Ce module gérera l'interface de "Natural Language Querying" (Requêtage en langage naturel).

## Survol
Il permettra de transformer des phrases en langage naturel provenant de l'utilisateur en requêtes ou actions valides.

## Fonctionnalités (Futures)
- Traduction de requêtes utilisateur en intentions métiers.
- Génération de code SQL via des templates paramétrés, sécurisés et restreints (`SELECT` uniquement).
- Validation de l'isolation par tenant et blocage de toute requête non autorisée.
- Interface avec l'Agent Sage pour récupération de données.
