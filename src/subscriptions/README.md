# Subscriptions Module - InsightSage

Ce module a pour vocation de gérer les abonnements des clients d'InsightSage (ex: Startup, PME, Business, Enterprise).

## Rôle
Lorsqu'InsightSage déploiera ses offres payantes et sa facturation, ce module sera le point central pour :
- Lister les plans d'abonnements disponibles.
- Restreindre l'accès à certaines fonctionnalités (limite de requêtes de l'Agent, nombre maximal de Widgets dans le Store, etc.) selon le plan rattaché à l'`organizationId`.
- Gérer les montées/descentes en gamme et la facturation.
