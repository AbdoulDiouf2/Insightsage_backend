# Onboarding Module - InsightSage

Ce module a pour but de fournir le tunnel de configuration initiale lorsqu'un client instancie son propre espace sur InsightSage.

## Enjeu
- Étape 1 : Assistant de choix d'abonnements.
- Étape 2 : Configuration d'une nouvelle organisation.
- Étape 3 : Spécification des détails de connexion (type de base Sage).
- Étape 4 : Définition des KPIs initiaux du Dashboard par défaut.
- Étape 5 : Invitation des premiers utilisateurs (collaborateurs).
